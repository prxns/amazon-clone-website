# Amazon Clone (Flask + SQLite)

A full-stack Amazon clone built using Flask, SQLite, HTML, and CSS.  
This project replicates core Amazon functionality including category browsing and dynamic product rendering from a database.

---

## Live Features

- Homepage with Amazon-style category cards
- Dynamic category pages
- Products loaded from SQLite database
- Flask backend with routing
- Template rendering using Jinja2
- Organized product image storage
- Clean and responsive UI layout
- Real product data and images

---

## Tech Stack

Frontend:
- HTML5
- CSS3

Backend:
- Python
- Flask

Database:
- SQLite

Tools:
- VS Code
- Git & GitHub

---

## How It Works

• Flask handles routing  
• SQLite stores product data  
• Jinja2 dynamically renders product pages  
• Images are loaded from organized folders  
• Category links load products dynamically  

---

## How to Run

1. Clone the repository
2. Open the project folder
3. Install & run flask server (app.py)
4. Open browser and go to http://127.0.0.1:5000
